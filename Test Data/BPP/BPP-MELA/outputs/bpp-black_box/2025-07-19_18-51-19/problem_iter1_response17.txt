1. Current heuristic returns uniform matrix, ignoring node attributes and constraints.
2. For bin packing, item size and bin capacity should influence decisions.
3. Create heuristic matrix where larger items are favored for earlier placement.
4. Use exponential decay to prioritize items that fit well in remaining capacity.}

```python
import numpy as np
def heuristics_v1(node_attr, node_constraint):
    #EVOLVE-START
    n = node_attr.shape[0]
    heuristic = np.zeros((n, n))
    for i in range(n):
        remaining_cap = node_constraint - np.cumsum(node_attr[:i])
        for j in range(n):
            if node_attr[j] <= remaining_cap[i]:
                heuristic[i,j] = np.exp(-node_attr[j]/remaining_cap[i])
    return heuristic
    #EVOLVE-END
```
