1.Current heuristic is trivial (all ones), offering no guidance for bin packing.
2.Should incorporate item sizes and capacity constraints.
3.Use size-based attraction: larger items should have higher heuristic values when paired with smaller items to fill bins efficiently.
4.Normalize to prevent dominance over pheromones.}

```python
import numpy as np
def heuristics_v1(node_attr, node_constraint):
    #EVOLVE-START
    n = node_attr.shape[0]
    sizes = node_attr[:,0]
    heuristic = np.zeros((n,n))
    for i in range(n):
        for j in range(n):
            if i != j:
                heuristic[i,j] = (sizes[i] + sizes[j]) / node_constraint
    return heuristic / np.max(heuristic)
    #EVOLVE-END
```
