1.Current heuristic is trivial (all ones), providing no guidance.
2.Should incorporate problem-specific info - bin capacity and item sizes.
3.Larger items should have higher heuristic values when paired with similar-sized items.
4.Use inverse of size difference to encourage similar items in same bin.
5.Normalize for consistent scaling.}

```python
import numpy as np
def heuristics_v1(node_attr, node_constraint):
    #EVOLVE-START
    n = node_attr.shape[0]
    sizes = node_attr[:,0]
    heuristic = np.zeros((n,n))
    for i in range(n):
        for j in range(n):
            if i != j:
                heuristic[i,j] = 1/(1 + abs(sizes[i]-sizes[j]))
    np.fill_diagonal(heuristic, 0)
    return heuristic/node_constraint
    #EVOLVE-END
```
