{thought process:
1. Current heuristic uses uniform weights (ones matrix) which provides no guidance.
2. Bin packing benefits from size-based heuristics.
3. Should incorporate both item sizes and remaining capacity.
4. Use inverse of size differences to favor fitting items well.
5. Add noise for exploration.}

```python
import numpy as np
def heuristics_v1(node_attr, node_constraint):
    #EVOLVE-START
    n = node_attr.shape[0]
    sizes = node_attr[:,0]
    h = np.abs(np.subtract.outer(sizes, sizes)) / node_constraint
    h = 1 / (1 + h)  # Inverse relationship
    np.fill_diagonal(h, 0)  # No self-transitions
    h += 0.1 * np.random.rand(n, n)  # Exploration
    return h
    #EVOLVE-END
```
