1. The error occurs because the code assumes node_attr is a 2D array (indexed with [:,0]), but it's actually 1D
2. We need to handle both cases: when node_attr is 1D (single feature) or 2D (multiple features)
3. For the heuristic calculation, we only need the size attribute (first column if 2D, or the array itself if 1D)
4. The random exploration factor and diagonal zeroing logic remains the same}

```python
import numpy as np
def heuristics_v2(node_attr, node_constraint):
    #EVOLVE-START
    if node_attr.ndim == 1:
        sizes = node_attr
    else:
        sizes = node_attr[:,0]
    n = len(sizes)
    heuristic = np.outer(sizes, sizes) / (node_constraint**2)
    heuristic += 0.1 * np.random.rand(n,n)  # exploration factor
    np.fill_diagonal(heuristic, 0)  # avoid self-packing
    return heuristic
    #EVOLVE-END
```
