1. The error occurs because node_attr is a 1-dimensional array but the code tries to index it as 2-dimensional with node_attr[:,0]
2. We need to handle both cases where node_attr could be 1D or 2D
3. For 1D arrays, we can use the array directly as weights
4. For 2D arrays, we'll take the first column as weights
5. The rest of the heuristic calculation remains the same
6. Normalization ensures the heuristic values are between 0 and 1}

```python
import numpy as np
def heuristics_v2(node_attr, node_constraint):
    #EVOLVE-START
    if len(node_attr.shape) == 1:
        weights = node_attr / node_constraint
    else:
        weights = node_attr[:,0] / node_constraint
    heuristic = np.outer(weights, weights)
    return heuristic / heuristic.max()
    #EVOLVE-END
```
