1. The error occurs because node_attr is a 1-dimensional array but the code tries to index it as 2-dimensional with node_attr[:,0]
2. We need to first check the dimensionality of node_attr and handle both cases (1D and 2D arrays)
3. If it's 1D, we can use it directly as sizes
4. If it's 2D, we'll take the first column as sizes

import numpy as np
def heuristics_v2(node_attr, node_constraint):
    #EVOLVE-START
    # Handle both 1D and 2D node_attr cases
    if node_attr.ndim == 1:
        sizes = node_attr
    else:
        sizes = node_attr[:,0]
    
    n = len(sizes)
    heu = np.zeros((n,n))
    for i in range(n):
        for j in range(n):
            heu[i,j] = 1 - abs(sizes[i] - sizes[j])/node_constraint
    return np.maximum(heu, 0)
    #EVOLVE-END
