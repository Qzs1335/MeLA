{thought process:
1. Current heuristic uses simple inverse distance which may oversimplify path choices
2. Adding exponential decay can emphasize closer cities more
3. Incorporating a small constant prevents division by zero
4. Normalization could help balance the scale of heuristic values}  

```python
import numpy as np 
def heuristics_v1(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-10  # prevents division by zero
    alpha = 0.5      # controls decay rate
    normalized_dist = distance_matrix / np.max(distance_matrix)
    return np.exp(-alpha * normalized_dist) + epsilon
    #EVOLVE-END       
    return 1 / distance_matrix
```
