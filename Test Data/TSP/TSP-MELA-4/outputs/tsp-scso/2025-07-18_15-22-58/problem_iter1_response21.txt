{thought process:
1. Current heuristic uses simple inverse distance, lacking exploration-exploitation balance.
2. Adding logarithmic scaling improves sensitivity to small distances.
3. Incorporating exponential decay helps handle outlier distances.
4. Normalization ensures consistent heuristic values across different problem scales.}

```python
import numpy as np
def heuristics_v1(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-10  # Prevent division by zero
    scaled = np.log(1 + distance_matrix)
    heuristic = np.exp(-0.5 * scaled)
    heuristic = heuristic / (heuristic.max() + epsilon)
    #EVOLVE-END       
    return heuristic
```
