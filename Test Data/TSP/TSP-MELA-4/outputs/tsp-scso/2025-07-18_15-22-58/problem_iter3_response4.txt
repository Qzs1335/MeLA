1. The current version combines exponential decay and inverse distance well but can be more numerically stable.
2. The epsilon (1e-5) prevents division by zero but could be smaller (1e-10) for better precision.
3. The adaptive scaling is good but could use median for outlier robustness.}  
```python
import numpy as np 
def heuristics_v2(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-10  # improved precision
    adaptive_scale = np.median(distance_matrix)  # more robust scaling
    decay = np.exp(-distance_matrix/(adaptive_scale + epsilon))
    inv_dist = (1/(distance_matrix + epsilon))**2.0
    return decay * inv_dist
    #EVOLVE-END       
    return 1 / distance_matrix
```
