1. Current version combines exponential decay and inverse distance well
2. Need to add distance-based weighting for better exploration-exploitation balance
3. Introduce dynamic exponent β that adapts to distance distribution
4. Maintain numerical stability with ε while enhancing gradient sensitivity}

```python
import numpy as np 
def heuristics_v2(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-5
    adaptive_scale = np.mean(distance_matrix)
    dynamic_beta = 2.0 - (distance_matrix / (np.max(distance_matrix) + epsilon))  # distance-adaptive exponent
    decay = np.exp(-distance_matrix/(adaptive_scale + epsilon))
    inv_dist = (1/(distance_matrix + epsilon))**dynamic_beta
    return decay * inv_dist
    #EVOLVE-END       
    return 1 / distance_matrix
```
