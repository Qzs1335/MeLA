{thought process:
1. Current version combines exponential decay and inverse distance well but lacks exploration control.
2. Adding a small random component (5% weight) could improve exploration.
3. Maintain existing adaptive scaling (ε=1e-5) for stability.}     
```python
import numpy as np 
def heuristics_v2(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-5
    adaptive_scale = np.mean(distance_matrix)  
    decay = np.exp(-distance_matrix/(adaptive_scale + epsilon))
    inv_dist = (1/(distance_matrix + epsilon))**2.0
    random_component = 0.05 * np.random.rand(*distance_matrix.shape)
    return (1-random_component)*decay*inv_dist + random_component
    #EVOLVE-END       
    return 1 / distance_matrix
```
