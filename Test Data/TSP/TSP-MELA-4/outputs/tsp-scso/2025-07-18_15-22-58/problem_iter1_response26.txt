1. Current heuristic uses simple inverse distance which may oversimplify TSP complexity.
2. Adding exponential decay can emphasize nearby cities more strongly.
3. Incorporating a small constant prevents division by zero and adds exploration.
4. Logarithmic scaling could better handle distance variations.}

import numpy as np 
def heuristics_v1(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-5
    scaled_dist = np.log(distance_matrix + epsilon)
    return np.exp(-0.5 * scaled_dist)
    #EVOLVE-END       
    return 1 / distance_matrix
