2. We need to handle cases where distance_matrix contains zeros or very small values that could cause numerical instability
3. Adding a small epsilon helps prevent log(0) but we should also clip the values to avoid extreme results

```python
import numpy as np
def heuristics_v2(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-10
    # Clip the distance matrix to avoid extreme values
    clipped_distances = np.clip(distance_matrix, epsilon, None)
    # Apply transformation and ensure all values are finite and positive
    transformed = np.power(np.log(clipped_distances), -1.5)
    # Normalize to avoid numerical issues with the Categorical distribution
    normalized = transformed / np.sum(transformed, axis=1, keepdims=True)
    return normalized
    #EVOLVE-END
```
