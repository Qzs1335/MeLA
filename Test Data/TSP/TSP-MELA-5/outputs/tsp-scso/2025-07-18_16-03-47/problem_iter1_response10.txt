1. Current heuristic uses simple inverse distance which may overlook local optima.
2. Adding exponential decay can emphasize nearby cities more.

4. Combining both creates non-linear distance sensitivity.}

import numpy as np
def heuristics_v1(distance_matrix):
    #EVOLVE-START
    return np.exp(-0.5 * distance_matrix) * np.log(1 + 1/distance_matrix)
    #EVOLVE-END
    return 1 / distance_matrix
