2. We can improve by using a softer transformation (sqrt) to moderate non-linearity while maintaining stability.
3. Keep epsilon but apply it after sqrt to balance proximity weighting and numerical safety.}

import numpy as np 
def heuristics_v2(distance_matrix):
    #EVOLVE-START
    epsilon = 1e-15
    return 1 / (np.sqrt(distance_matrix) + epsilon)
    #EVOLVE-END       
    return 1 / distance_matrix
